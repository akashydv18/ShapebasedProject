# ShapeLearningGame

An interactive **Java-based educational game** designed for students of **classes 6 to 12** to improve their understanding of **2D and 3D shapes** in a fun and engaging way.

The project provides an interactive UI where students can click on shapes, explore their properties, and strengthen geometry concepts through visual learning.

---

## 🚀 Features

- Interactive educational game for students
- Supports learning of:
  - **2D Shapes**
  - **3D Shapes**
- Clickable shape-based user interface
- Visual and engaging learning experience
- Helps students understand shape identification and concepts
- Structured using **Object-Oriented Programming (OOP)** principles
- Scalable and maintainable code design

---

## 🎯 Project Objective

The main goal of this project is to make **geometry learning more interactive and enjoyable** for school students.

This game is specially designed to help students:
- Recognize different shapes
- Understand basic shape properties
- Improve conceptual learning through interaction
- Learn visually rather than only through theory

## 🧠 OOP Concepts Used

This project is built using important **Object-Oriented Programming concepts**, including:

### 1. Inheritance
Used to create a hierarchy between different shape types.

### 2. Method Overloading
Used to provide multiple ways to work with shape properties.

### 3. Method Overriding
Used to customize behavior for different shapes.

### 4. Upcasting
Used to handle different shape objects using a common parent reference.

### 5. Has-a Relationship
Used to manage composition and improve modular design.

These concepts make the code:
- reusable
- scalable
- maintainable
- easy to extend with more shapes in the future
